# Toji Shooter (Scratch Game)

A simple 2D shooter built in Scratch.

Play as Toji (me), fire projectiles, and eliminate incoming zombies.
Core combat loop with movement, shooting, collision detection, and sound effects.

## Gameplay

- Move Toji using keyboard controls
- Shoot projectiles
- Eliminate zombies
- Audio feedback for actions

## Features

- Player movement system
- Projectile spawning
- Collision detection
- Basic enemy behavior
- Sound integration

## How to Run

Option 1 — Play in Browser (Recommended)

- Open the Scratch link above
- Click Green Flag to start

Option 2 — Run Locally

- Download the .sb3 file
- Open it using:
    - Scratch Desktop
    - OR upload it to the Scratch website editor
    - Click Green Flag to start
